# Weather Forecasting Analysis

## Overview
This project analyzes historical weather data to identify
patterns in temperature, humidity, and rainfall using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Structure
data/ - dataset
notebooks/ - analysis notebook

## How to Run
1. Install libraries

pip install pandas numpy matplotlib

2. Open the notebook

weather_analysis.ipynb
